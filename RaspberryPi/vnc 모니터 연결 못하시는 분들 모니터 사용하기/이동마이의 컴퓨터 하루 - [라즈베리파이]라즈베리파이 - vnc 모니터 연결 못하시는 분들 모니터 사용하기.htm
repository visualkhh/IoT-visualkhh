[라즈베리파이]라즈베리파이 - vnc 모니터 연결 못하시는 분들 모니터 사용하기
Raspberry Pi 13. 01/03


크리에이티브 커먼즈 라이선스
이 저작물은 크리에이티브 커먼즈 코리아 저작자표시-비영리-동일조건변경허락 2.0 대한민국 라이선스에 따라 이용하실 수 있습니다.
주의 - 발열이 심하게납니다.

안녕하세요.^^ 오늘은 라즈베리 파이를 이용한 쉬운 Xwindows였나? vnc로 쉽게 라즈베리파이를 쓸수있는 방법을 알려드리겠습니다.


필자는 이미 다 설치해서 중간부분은 못 알려드리니 양해부탁드려요.



일단 라즈베리파이에 로그인후 저렇게 입력해 주세요

sudo apt-get install tightvncserver

그후 파일을 찾아보는걸탠데요(?) 그후에

필자는 이미 설치해서 중간부분이 안나왔지만
